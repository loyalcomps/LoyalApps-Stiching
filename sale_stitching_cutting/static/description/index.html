Overview

This App can help you to supervise and handle Tailoring orders with ease. One can easily create sales quotation and easily get processed and scheduled the work. With simple processing steps, one can complete all the tasks with a few clicks with odoo ERP implementation.Allowing Sales person to listing out to choose users for Cutting and Stitching. By choosing the two user group like cutting and stitching makes the work easier and get noticed the completion of works. This keeps you a eye on your sales order consolidating cutting and stitching users against the sales orders and their work status.

Configuration

  • No additional configuration required

Features

  • Allows to choose cutting and stitching task from Sales Quotation
  • Alloting the Users to the specified tasks
  • Choosing the first operation as cutting and Selecting the Appropriate User
  • Once the Cutting is done, You have a button validating that the cutting task has been done
  • Validation for delivery of products only after Cutting and Stitching
  • Choosing the next task as Stitching and Allotes the required user
  • Choosing the task Stitching has been done and gets validated
  • Delivery initiates only after both tasks Cutting and Stitching has been done
  • Allows you to check the status of the sales order in the List view
  • Reports can be generated based on Users in a time frame
  • Based on Pending and Completed status, Reports will be generated
